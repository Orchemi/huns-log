## 설치 과정

`[nodejs.org](https://nodejs.org/ko/download/current/)`의 다운로드 페이지에 들어가면, LTS 버전과 현재 최신 버전으로 나누어 다운로드 받을 수 있다.

우선 LTS는 오류를 다듬고 다듬어 나름 안정적인 상태의 버전으로 대다수의 사용자에게 권한다고 하여 다운로드 받았지만 더 최근 버전이 있다고 오류가 뜨며 설치창이 중단되길래 그냥 최신 버전으로 깔아버렸다.

<br />

`![image](https://user-images.githubusercontent.com/86189596/153709948-fdc434b2-9163-42f7-b9e5-ddcfdc5e8acd.png)`

Next, Next를 누르면 되는데 설치할 항목들 목록이 나온다. 기본적으로 Node.js를 설치하면 npm package manager가 함께 설치되고 PATH에 기본 경로가 추가된다. 역시나 별 건드릴 필요 없이 그대로 Next를 누르며 진행한다.

별 단계도 없이 성공적으로 다운로드가 종료된다.

<br />

## 설치 확인

node.js는 터미널에서 쉽게 확인할 수 있다. [window] + [R] 키를 누르고 [cmd]를 눌러 커맨드창을 띄운다. 그리고 아래의 명령어를 입력한다.

```
node -v
```

node.js의 version을 알려달라는 얘기이다. 나는 최신 버전인 `v17.5.0`이 나왔다.

다시 command line에 node를 입력하면 `Welcome to Node.js v17.5.0`이라며 멋있게 환영을 해주고 node.js를 실행할 수 있게 된다.

<br />

console에 입력하는 javascript 문법처럼 입력해보자.

```
var i = 1          -> 변수 i에 1을 할당한다.
var i = 2          -> 변수 i에 2를 재할당한다.
var j = 5          -> 변수 j에 5를 할당한다.
console.log(i*j)   -> console에 i*j의 값을 출력한다.
```

이 과정을 마치면 아래와 같은 결과를 맞이할 수 있다.

`![image](https://user-images.githubusercontent.com/86189596/153709880-1ce30d70-eb0e-4620-a4d2-a8efc1e366fe.png)`

역시나 console에서 한 것처럼 `console.log(i*j)` 가 아닌 `i*j`만 입력해도 값은 나온다.

## javascript 파일 실행

`![image](https://user-images.githubusercontent.com/86189596/153710332-6f899d58-f8d7-4cff-b5d9-1f07549ca59b.png)`

개인정보에 관련된 것들은 지우겠다.

js 파일이 있는 폴더에 위치한 뒤, `$ node [파일명.js]`을 입력하여 js파일을 실행한다. 나는 hello.js 파일을 실행하기 위해 `$ node hello.js`를 입력하였고, console에 출력되듯이 터미널에 hello world가 출력됨을 확인할 수 있었다.
