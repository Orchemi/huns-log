## 1. 예제

```python
# 예제 1. 애완동물을 소개해주세요.

print("우리집 강아지의 이름은 연탄이에요.")
print("연탄이는 4살이며, 산책을 아주 좋아해요.")
print("연탄이는 어른일까요? True")
```

<br />

## 2. 변수 지정

```python
animal = "강아지"  # 문자열이라서 큰 따옴표 처리
name = "연탄이"
age = 4  # 숫자형이라서 따옴표 없이 바로
hobby = "산책"
is_adult = age >= 3  # boolean 형태
```

<br />

## 3. 변수 변경

1. 문자열 변수 문자열 사이에는 "+"로 연결
2. 숫자형 / boolean 형태는 str(변수)를 통해 문자열로 변경
3. '+' 대신 반점(,)을 연결문자로 사용한다면 숫자형 / boolean 형도
   str(변수) 사용 없이 변수 자체로 사용 가능  
   ※ 단, 반점으로 구분한 변수와 문자열 사이는 공백이 한 칸 생겨 출력됨

```python
print("우리집 " + animal + "의 이름은 " + name + "에요.")
print(name + "는 " + str(age) + "살이며, " + hobby + "을 아주 좋아해요.")
print(name + "는 어른일까요?" + str(is_adult))
```

<br />

## 4. 변수 중복

> 상위 코드에서 지정한 변수여도 하위 코드에서 재입력하면, 하위 코드에서 입력한 것으로 최신화

```python
animal = "강아지"
name = "연탄이"
age = 4
hobby = "산책"
is_adult = age >= 3

print("우리집 " + animal + "의 이름은 " + name + "에요.")
hobby = "공놀이"
print(name + "는 " + str(age) + "살이며, " + hobby + "을 아주 좋아해요.")  # "공놀이을 아주 좋아해요." 가 된다.
print(name + "는 어른일까요?" + str(is_adult))
```
