## 파이썬에서의 주석 처리

1. '#'으로 처리한다.
2. 작은 따옴표(') 3개 로 시작하고, 3개로 닫아서 여러 문장을 단체 주석
3. 여러 문장을 선택한 후 `<ctrl + / >` 눌러서 여러 문장을 단체 주석 설정/해지

```python
# 주석1

'''
여러 문장
주석 처리
작은 따옴표
범위 내에
내용 삽입
'''
```
