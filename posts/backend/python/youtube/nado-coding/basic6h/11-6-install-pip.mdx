# pip()

- 이미 잘 만들어진 package를 필요한 곳에 가져다 쓰는 것이 필요

<br />

## 가. pypi(python package index) 사용 방법

- 'beautifulsoup' package를 install해서 사용해보자.
- 웹페이지의 스크래핑을 쉽게 해주는 package

```markdown
1. google에 'pypi'를 치고 https://pypi.org/ 로 들어간다.
2. 검색창에서 'beautifulsoup'를 검색한다. (https://pypi.org/project/beautifulsoup4/)
3. pip install beautifulsoup4 copy
4. terminal에 paste
```

<br />

## 나. 코드의 사용

```python
from bs4 import BeautifulSoup
soup = BeautifulSoup("<p>Some<b>bad<i>HTML")
print(soup.prettify())
```

<br />

## 다. pip 명령어

- terminal에서 사용할 수 있는 pip 명령어

### 1) 현재 설치되어있는 pip 리스트 확인

```python
pip list
```

<br />

### 2) beautifulsoup4 package의 세부정보를 확인하기

```python
pip show beautifulsoup4
```

<br />

### 3) package의 버전을 업그레이드하기

```python
pip install --upgrade beautifulsoup4
```

<br />

### 4) package 제거

```python
pip uninstall beautifulsoup4
```
