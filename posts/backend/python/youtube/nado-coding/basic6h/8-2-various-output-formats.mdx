# 다양한 출력 포맷

## 예제 1

- 빈 자리는 빈 공간으로 두고, 오른쪽 정렬을 하되, 총 10자리 공간 확보

```python
print("{0: >10}".format(500))

# > : 오른쪽 정렬
# 10 : 10자리 공간 확보

# 출력값 :        500
```

<br />

## 예제 2

- 양수일 때는 + 표시, 음수일 때는 - 표시

```python
print("{0: >+10}".format(500))
print("{0: >+10}".format(-500))
```

```python
# 출력값 :
#      +500
#      -500
```

```markdown
예제 1과 2의 차이점  
예제 1은 양수인 경우 부호 없이 그냥 찍히지만,  
예제 2는 양수인 경우 + 부호가 앞에 있다는 점
```

<br />

## 예제 3

- 왼쪽 정렬하고, 빈칸을 \_로 채운다.

```python
print("{0:_<+10}".format(500))

# 출력값 : +500______
```

<br />

## 예제 4

- 큰 숫자의 경우, 3자리마다 comma(,)을 찍어준다.

```python
print("{0:,}".format(1000000000))

# 출력값 : 1,000,000,000
```

<br />

## 예제 5

- 큰 숫자의 경우, 3자리마다 comma(,)을 찍어주고 부호까지 붙인다.

```python
print("{0:+,}".format(1000000000))
print("{0:+,}".format(-1000000000))

# 출력값 :
# +1,000,000,000
# -1,000,000,000
```

<br />

## 예제 6

- 3자리마다 comma(,)를 찍어주고, 부호를 붙이고, 자릿수도 확보한다.
- 빈 자리는 ^ 으로 채운다.
- 왼쪽 정렬, 30자리 확보

```python
print("{0:^<+30,}".format(10000000000))
# 출력값 : +10,000,000,000^^^^^^^^^^^^^^^
```

<br />

## 예제 7

- 소숫점 출력

```python
print("{0:f}".format(5/3))
# 출력값 : 1.666667

print("{0:.2f}".format(5/3))
# 소수점 셋째 자리에서 반올림
# 출력값 : 1.67
```
