## 가. 숫자형

```python
# 숫자형
print(5)  # 숫자는 큰 따옴표 없이
print(-10)  # 음수도 가능
print(3.14)  # 실수도 가능


# 숫자 연산
print(3+5)
print(2*3)
print(3*(3+1))  # 우선순위
print(2**3)  # 2^3 제곱형
```

<br />

## 나. 문자열

```python
# 문자열
print('풍선')  # 작은 따옴표
print("나비")  # 큰 따옴표
print("ㅋ"*9)  # 문자열과 숫자 연산도 가능
```

<br />

## 다. Boolean 형

```python
# Boolean : 논리(참/거짓)
print(5 > 10)  # False
print(5 < 10)  # True
print(True)  # True
print(not True)  # True의 반대 : False
print(not (5 > 10))  # False의 반대 : True
```
