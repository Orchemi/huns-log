# 패스(Pass)

- 코드가 미완성이어도 오류 없이 다음 코드로 넘어가는 것

## 상황 1. 미완성 코드

```python
def game_start():
    print("[알림] 새로운 게임을 시작합니다.")

def game_over():
```

그리고 위의 두 함수를 호출해본다.

```python
# 함수 호출
game_start()
game_over()
```

출력결과는 다음과 같다.

```markdown
출력값 : 오류 발생
game_start()
^
IndentationError: expected an indented block
```

<br />

## 상황 2. Pass 사용

```python
def game_start():
    print("[알림] 새로운 게임을 시작합니다.")

def game_over():
    pass
```

그리고 위의 두 함수를 호출해본다.

```python
# 함수 호출
game_start()
game_over()
```

출력결과는 다음과 같다.

```markdown
[알림] 새로운 게임을 시작합니다.
```

즉, **구문이 완성되지 않아 오류가 출력되어야 할 상황이더라도,
pass를 함으로써 오류를 출력하지 않는 것이다.**
