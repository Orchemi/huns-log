## 1. 한 문장의 문자열 변수

- 문자열을 작은 따옴표(')로 감싸준다.
- 문자열을 큰 따옴표(")로 감싸준다.

```python
sentence = '나는 소년입니다'
print(sentence)

sentence2 = "파이썬은 쉬워요"
print(sentence2)
```

<br />

## 2. 여러 문장의 문자열

- 여러 문장의 문자열을 큰따옴표 3개(""") 안에 넣는다.

```python
sentence3 =
"""나는 소년이고,
파이썬은 쉬워요"""

print(sentence3)
```
