## 1. 함수 정의 방법

```markdown
함수명():
하위 코드
```

- 예시

```python
def open_account():
    print("새로운 계좌가 생성되었습니다.")
```

<br />

## 2. 함수 사용 방법

- 그냥 함수명()을 쓴다

```python
def open_account():
    print("새로운 계좌가 생성되었습니다.")

open_account()
# 출력값 : 새로운 계좌가 생성되었습니다.
```
