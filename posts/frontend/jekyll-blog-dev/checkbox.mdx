# 체크박스를 만들어보자

## 1. 활용법

> TIL에서 To Do List를 잘 수행했는지 확인할 때 사용해보자.

## 2. 어떻게 만드나요?

> 대괄호 사이에 공백을 하나 넣어준다.

> 수행한 Task에 대해서는 대괄호 안에 영어 X를 넣어준다.
> 소문자와 대문자는 상관 없다.

```md
- [ ] contents
- [x] contents
```
