## 0. 들어가기

- 모든 일정은 전년도 일정에 의거하여 기간을 유추하였으며, 추후 정확한 기간에 대한 안내, 그리고 변경되는 사항들에 대해 추가할 예정.

<br />

## 1. 소프트웨어 마에스트로

### 가. 접수기간 및 주요일정

```markdown
1. 공고 : 1월 중순 ~ 2월 중순
2. 서류 검토 : 2월 중순 ~ 2월 말
3. 1차 전형(코딩테스트) : 2월 말(서류 통과 직후)
4. 2차 전형(코딩테스트) : 3월 중순
5. 3차 전형(심층 면접) : 3월 말
6. 교육기간 : 6월 ~ 11월(약 5개월)
```

<br />

### 나. 지원자격

```markdown
학력, 전공, 경력 : 제한 없으나 SW개발(언어)능력을 갖춘 자

※ 지원할 수 없는 경우 - 연수 기간 중 재직자, 취업 확정자, 군복무(산업기능요원 포함) 중인 자 - 연수 기간 중 정부에서 추진하는 취·창업 지원 프로그램 등에 참여중이거나, 참여 예정인 자
```

<br />

### 다. 연수생 특전

```markdown
1. 장학금 : 매월 100만원
2. 노트북 등 IT기기 구입비 지원(1회)
3. 프로젝트 개발비(팀별 프로젝트 활동비)
4. 교육 및 개발공간
5. 미취업자 대상 인턴십 연계, 취업 컨설팅 지원
```

<br />

### 라. 기타

```markdown
- 1월 말(응시원서 접수기간 중)에 선발 및 교육과정 온라인 설명회가 있으니 참고
```

## 2. SSAFY

### 가. 접수기간 및 주요일정

```markdown
1. 접수일 : 10월 말 ~ 11월 초
2. 적성진단 : 11월 중순
3. 인터뷰 : ?
4. 교육기간 : 1월~12월(12개월)
```

<br />

### 나. 지원자격

```markdown
- 국내외 4년제 대학(학사 이상) 졸업자 및 졸업 예정자
- 전공무관
- 미취업자 대상
- 교육시작일 교육 입과 가능한 자
- 교육기간 중 통학 가능한 자(별도 숙소 제공 없음)
```

<br />

### 다. 연수생 특전

```markdown
- 장학금 : 매월 100만원
- 우수 연수생 : 해외연구소 실습기회 부여
- 개인별 진로 컨설팅, Job Fair 등 취업지원 서비스 제공
- 삼성 SW 역량 테스트 응시기회 제공
```

## 3. 우아한 테크코스

### 가. 접수기간 및 주요일정

```markdown
1. 서류접수 : 10월 말 ~ 11월 초
2. 온라인 코딩테스트 : 11월 초(서류 접수 직후)
3. 1차 합격자 발표 : 11월 중순
4. 프리코스 : 11월 말 ~ 12월 중순
5. 최종 코딩테스트 : 12월 말
6. 최종 선발 : 1월 초
7. 교육기간 : 2월 초 ~ 11월 말(약 10개월)
```

<br />

### 나. 지원자격

```markdown
- 프로그래밍에 대한 기본 지식과 경험이 있는 분
- 함수, 변수, 데이터 타입, 조건문, 반복문을 활용해 프로그래밍 구현이 가능한 수준
```

## 4. 네이버 부스트캠프

### 가. 접수기간 및 주요일정

```markdown
※ 1년에 1기수로 운영되며, 매년 중반기부터 모집, 하반기 교육 진행인 것 같다.

1. 접수일 : 5월 중순 ~ 6월 중순
2. BAT AI테스트 : 12월 중순(서류 통과 직후)
3. 1차 코딩테스트 : 6월 말
4. 2차 코딩테스트 : 7월 초
5. 교육기간 : 7월 ~ 12월(약 5개월)
```

<br />

### 나. 지원자격

```markdown
- 국민내일배움카드 신규발급대상자
- 반기 풀타임(월~금, 10:00~19:00) 참여 가능한 자
- 개발 경력 2년 미만
```

<br />

### 다. 연수생 특전

```markdown
- 장학금 : X
- 웰컴킷이 있음
- 네트워킹데이를 통해 여러 기업의 인턴십/정규직의 기회 제공
```

<br />

### 라. 기타

```markdown
위치 : 서울(코로나 없으면 오프라인) / 온라인(코로나 있으면 온라인)
```
